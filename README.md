# SubMan - Система управления подписками

## Описание проекта

Веб-приложение для учета и управления подписками на цифровые сервисы. Позволяет пользователям централизованно отслеживать все свои активные подписки, их стоимость и сроки оплаты, а администраторам - управлять подписками и анализировать активность пользователей.

## Функционал

### Для подписчиков

- **Авторизация и регистрация**
  - Регистрация и аутентификация пользователей
  - Авторизация в роли администратора или подписчика
  - Смена пароля

- **Личный кабинет**
  - Просмотр и редактирование данных профиля
  - Просмотр истории покупок и истории платежей (дата, сумма, статус)
  - Просмотр информации о личных подписках:
    - Название подписки
    - Статус (активна, приостановлена, отменена, просрочена)
    - Стоимость
    - Период действия подписки (месяц, год)
    - Дата окончания подписки
    - Сумма и дата следующего платежа при продлении
  - Просмотр уведомлений (при необходимости продления, при проблемах с платежами)
  - Управление статусом подписки:
    - Оформление новой подписки
    - Временная приостановка подписки
    - Прекращение подписки (с указанием причины)
    - Восстановление отмененной подписки

- **Каталог и контент**
  - Просмотр каталога цифрового контента и тарифных планов
  - Просмотр статей и часто задаваемых вопросов (FAQ)
  - Возможность оставить оценку удовлетворенности и отзыв
  - Применение промокодов

- **Поддержка**
  - Онлайн-чат с администратором

### Для администратора

- **Аналитическая панель (дашборд)**
  - Просмотр общего количества активных пользователей
  - Распределение подписок по категориям цифрового контента
  - Мониторинг подписок:
    - Новые подписки
    - Отмененные подписки
    - Оплаченные подписки
    - Истекающие подписки
  - Статусы платежей (успешные платежи, ошибки в оплате)

- **Управление**
  - Управление подписчиками
  - Управление подписками (создание, редактирование, удаление, приостановка)
  - Управление тарифными планами
  - Формирование промокодов и условий скидок

- **Отчетность**
  - Отчет об активности пользователей за выбранный период/поставщика цифрового контента
  - Отчет по подпискам в разрезе временного периода
  - Детализация по подписчикам
  - Счета установленного образца
  - Экспорт данных в формат PDF, MS Excel, MS Word
